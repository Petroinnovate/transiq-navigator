🔧 TROUBLESHOOTING QUICK REFERENCE
═══════════════════════════════════════════════════════════════════════════════════

Problem: "Python not found"
→ Run: python --version
→ If fails: Install from https://www.python.org/

Problem: "Node/Bun not found"  
→ Run: node --version  (or bun --version)
→ If fails: Install from https://nodejs.org/ or https://bun.sh/

Problem: "Port 8001 already in use"
→ Run: netstat -ano | findstr 8001
→ Kill the process or change port

Problem: "Backend won't start"
→ Check: Python version (3.8+)
→ Check: Terminal output for error messages

Problem: "WebSocket fails"
→ Ensure: Backend is actually running
→ Check: Firewall settings
→ Try: Restarting both services
